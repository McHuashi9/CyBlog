## 🎯 当前工作流程

### 写作发布流程：
```bash
# 一键创建和发布新文章
./new-post-and-publish.sh "文章标题"
```
# 流程：
1. 创建 Markdown 文章模板
2. 自动打开编辑器等待编辑
3. 提交更改到 Git
4. 推送到 GitHub 触发 CloudFlare 构建
5. 全球 CDN 自动部署

### 更新现有文章：
# 流程：
1. 使用喜欢的编辑器直接编辑 我的文章.md
2. **运行更新脚本**：
```bash
$ ./update-post.sh "我的文章"
 ```

 ### 删除现有文章：
 **运行更新脚本**：
```bash
$ ./delete-post.sh "我的文章"
 ```
